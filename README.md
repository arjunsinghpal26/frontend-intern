# Frontend Intern Projects 🚀

A collection of frontend development projects built during my internship journey. Each project showcases modern UI/UX design principles, responsive layouts, and interactive web experiences.

---

## 📁 Projects

### 🗓️ Calendar App
An interactive, high-fidelity wall calendar with:
- Day-range selection
- Month-based theming
- Dark mode support
- Smooth page-flip animations
- Notes section

📂 Located in: `calendar-app/`

---

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Vanilla CSS with animations & transitions
- **JavaScript** — Vanilla JS for interactivity
- **React / Next.js** — For component-based projects

